# Tiny BSCNet – Lightweight Semantic Segmentation Model

## Overview
This project implements a lightweight semantic segmentation model using PyTorch.

## Key Features
- Custom encoder-decoder architecture
- Context module (ELPPM)
- Boundary refinement module (BFM)
- Synthetic dataset generation

## What I did
- Created custom dataset with pixel-level labels
- Built full training pipeline
- Trained segmentation model
- Evaluated using IoU and pixel accuracy
- Visualized predictions and object detection

## Tech Stack
- Python
- PyTorch
- OpenCV
- NumPy

Project under improvement (working on real-world dataset)
